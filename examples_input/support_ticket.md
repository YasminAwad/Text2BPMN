# Customer Support Ticket Process

## Process Flow

1. Customer submits a support ticket through the web portal
2. System automatically categorizes the ticket based on keywords
3. If the ticket is marked as **urgent**, it gets escalated to senior support team
4. If the ticket is **standard priority**, it gets assigned to the general support queue
5. Support agent reviews the ticket and investigates the issue
6. Agent resolves the issue and updates the ticket
7. Customer receives notification that ticket is resolved
8. System sends follow-up satisfaction survey after 24 hours
